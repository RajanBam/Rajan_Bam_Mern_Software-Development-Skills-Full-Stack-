# Coursework — coding along with the module videos

These are the exercises I coded while working through the Software Development
Skills *Fullstack* module videos, before starting the Inkboard project. They are
small on purpose — the point was to get the fundamentals of each technology into
my fingers. My reflections on each are in the root `LEARNING_DIARY.md`.

| Folder | Video | What I practised |
| --- | --- | --- |
| `node-fundamentals/` | Node.js crash course | `path`, `url`, `fs`, `events`, and an HTTP server with **no** Express |
| `mongodb-notes/` | MongoDB crash course | CRUD with `mongosh` + the native driver |
| `express-notes/` | Express.js crash course | routing, middleware, validation |
| `react-notes/` | React crash course | components, state, hooks, events, routing |

Each folder has its own README with the commands to run it.
